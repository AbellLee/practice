# 刷题系统

一个现代化的在线刷题系统，支持题目练习、答题记录、统计分析等功能。

## ✨ 特性

- 📚 **刷题练习** - 丰富的题库支持
- 🎯 **快速答题** - 随机抽题、指定题量、进度显示
- 📊 **统计分析** - 详细的学习数据统计
- 📱 **移动端适配** - 完美的移动端体验
- 🌙 **深色模式** - 舒适的深色主题
- 📖 **错题本** - 自动收集错题
- ⭐ **收藏功能** - 收藏重要题目
- 🏆 **成就系统** - 学习成就和徽章

## 🚀 快速开始

### 环境要求

- Node.js >= 16.x
- Go >= 1.19

### 安装步骤

#### 1. 克隆项目
```bash
git clone <your-repo-url>
cd 刷题
```

#### 2. 安装前端依赖
```bash
cd frontend
npm install
```

#### 3. 安装后端依赖
```bash
cd backend
go mod download
```

#### 4. 启动后端服务
```bash
cd backend
go run .
```

后端服务将在 http://localhost:8080 启动

#### 5. 启动前端服务
```bash
cd frontend
npm run dev
```

前端服务将在 http://localhost:5173 启动

## 🎯 主要功能

### 1. 刷题练习
- 题目列表展示
- 题目详情查看
- 选择题/判断题支持
- 答案提交与判分
- 答案解析显示

### 2. 快速答题
- 随机抽题模式
- 指定题目数量（5/10/20/50/100 题）
- 答题进度显示
- 实时统计正确率
- 答题总结报告

### 3. 答题记录
- 历史答题记录
- 正确/错误筛选
- 答题时间统计

### 4. 统计分析
- 总答题数统计
- 正确率分析
- 分类统计
- 学习成就展示

### 5. 错题本
- 自动收集错题
- 错题复习
- 掌握状态标记

### 6. 收藏功能
- 题目收藏
- 收藏管理
- 快速访问

## 🎨 技术栈

### 前端
- React 18
- TypeScript
