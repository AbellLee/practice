# 移动端适配测试指南

## 快速启动

前端服务已启动在：http://localhost:5174/

## 测试方法

### 方法 1：浏览器开发者工具
1. 打开 Chrome/Edge 浏览器
2. 访问 http://localhost:5174/
3. 按 F12 打开开发者工具
4. 点击设备切换按钮（或按 Ctrl+Shift+M）
5. 选择不同设备进行测试

### 方法 2：真实设备测试
1. 确保手机和电脑在同一 WiFi 网络
2. 在手机浏览器访问：http://192.168.31.97:5174/
3. 测试各个功能
