## 🔍 Chrome DevTools 测试方法

### 1. 打开设备模拟
1. 按 `F12` 打开 DevTools
2. 按 `Ctrl+Shift+M` 或点击设备切换按钮
3. 选择预设设备或自定义尺寸

### 2. 常用设备预设
- iPhone SE (375 x 667)
- iPhone 12 Pro (390 x 844)
- iPhone 14 Pro Max (428 x 926)
- iPad Air (820 x 1180)
- iPad Pro 11" (834 x 1194)

### 3. 测试功能
- 触摸模拟
- 网络节流（2G/3G/4G）
- 设备方向切换
- 媒体查询调试
- 性能分析

## 📋 常见问题检查

### 布局问题
- [ ] 元素是否溢出屏幕
- [ ] 文本是否被截断
- [ ] 间距是否合理
- [ ] 对齐是否正确

### 交互问题
- [ ] 点击目标是否足够大（≥44px）
- [ ] 是否有误触风险
- [ ] 滚动是否流畅
- [ ] 动画是否卡顿

### 显示问题
- [ ] 字体大小是否可读
- [ ] 颜色对比度是否足够
- [ ] 图片是否自适应
- [ ] 图标是否清晰

### 性能问题
- [ ] 页面加载速度
- [ ] 滚动帧率
- [ ] 动画流畅度
- [ ] 内存使用

## 🎯 关键测试点

### 1. 触摸友好性
- 所有可点击元素 ≥ 44x44px
- 元素间有足够间距
- 无悬停依赖功能

### 2. 内容可读性
- 正文字体 ≥ 14px
- 行高适中（1.5-1.8）
- 对比度符合 WCAG 标准

### 3. 操作便捷性
- 重要操作在拇指热区
- 表单输入便捷
- 导航清晰直观

### 4. 性能表现
- 首屏加载 < 3 秒
- 交互响应 < 100ms
- 滚动帧率 > 50fps
